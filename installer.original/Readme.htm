README

Introduction

AS Remind is a minimal desktop application written in Java (read:widget) that links up with your entries on Google Calendar and informs you when an activity is coming up. In addition, it also allows you to make simple notes on the activities that you are doing on the track section of the application.

Installation

To install AS Remind, select the install.exe file and select various options in the installation process.
